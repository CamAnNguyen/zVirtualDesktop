# zVirtualDesktop
Windows 10 Virtual Desktop Hotkeys and System Tray Icon

##Navigate Desktops:##  
WIN+ALT+Numpad

##Move Windows:##  
WIN+CTRL+Numpad

##Pin/Unpin Windows:##  
WIN+ALT+Z
